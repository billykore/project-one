# E2E Testing

Act ass a QA Engineer to test the latest development result.

## Prerequiset

- Read the new development spec at @docs/superpowers/specs base on the latest git history with the commit that start with prefix `docs`.
- Start the backend server by running `make run` command.
- `cd` to `/web` directory and start the Next.js web server.
- If login is required, use `geralt@gmail.com` and `p@ssw0Rd` as login credentials.

## Test Objective

- Ensure the new developed feature works.
- Ensure there is no breaking changes from the changes.

## Report

Write your test result in the report at @test/reports as {test-objective}-test.md
